XML

  • 1) What is XML?

  • XML stands for eXtensible Markup Language, meaning that XML is a markup language that is very similar to HTML. Unlike HTML, XML tags are not predefined and XML is made to carry data.

  • 2) How is XML used?

  • XML is mostly used to markup Web and Mobile Applications. XML is able to exchange data between other systems like databases, other websites, or 3rd party applications. XML can send data back and forth with a web server.

JavaScript

  • 1) What is JavaScript?

  • JavaScript is a scripting language that lets you make a webpage dynamic by adding interactivity or complicated features.

  • 2) How is JavaScript related to HTML and CSS?

  • HTML is the content, CSS is the layout, and JavaScript is the behavior of webpages. For example you create a button with HTML, design it with CSS, and then use JavaScript to give that button some type of functionality.

  • 3) What are some common JS elements, properties, methods, and functions?

  • A JavaScript element is a variable that holds the data for a HTML element. JS lets you manipulate HTML elements by linking the data of a HTML element to the variable by it's ID, Class, or Tag Type.

  • A JavaScript function is a block of code that when executed, will perform a task. You can create your own functions that will do what you want, or you can use some helpful pre-built functions. An example of a pre-built function would be toString(), which can convert a variable's data type to a string.

  • JavaScript properties with DOM are the element values that you can get or set. For example you can get or change the text of a HTML element with innerHTML. There are also object properties which are variables within the scope of an object, but I don't think that is what you are looking for.

  • A JavaScript method is a function that does some type action to the DOM. For example you can use the innerHTML method to change the text of an HTML element. There are also object methods which are functions within the scope of an object, but again I don't think that is what you are looking for as well.
